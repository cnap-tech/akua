akua

One binary for cloud-native packaging. Render, verify, sign, publish. KCL-typed. Helm and Kustomize embedded as WebAssembly. Sandboxed by default.

Install (macOS / Linux)

$ curl -fsSL https://akua.dev/install | sh

Install (Windows)

> irm https://akua.dev/install.ps1 | iex

From source

$ cargo install --git https://github.com/cnap-tech/akua akua-cli

Pre-alpha. v0.1.0 ships four production-ready verbs; twenty-two more are wired behind the same command surface. See the repo for what is and isn't done yet.
